[Dawn of Worlds] solo Continent du Sud-Ouest

Kris 10 juin 2019 compte-rendu

Dawn of Worlds est un jeu de worldbuilding: on dessine vaguement les contours d'une carte, puis chaque joueur incarnant un immortel, on ajoute des éléments à chaque tour, on crée des espèces, puis on décide comment elles se comportent... A chaque tour, on obtient des points que l'on dépense pour influer sur le monde. La carte en illustration est le résultat de ma partie. J'y ai joué en solo, comme si j'incarnais deux joueurs différent (A et B). Les parties sont longues (plus de 3h et j'étais seule à jouer...)

Vous trouverez ici une aide de jeu pour jouer en français.

Compte Rendu:

Premier âge:

Tour n°1

A: 6 points B: 10 points

B:

  • 2 hexagones montagnes (6pts)
  • 1 hex forêt (3 pts)

A:

  • 1 hex forêt (3 pts) 1 hex collines (3pts)

Tour n°2:

A: 11+1=12 B: 8+1=9

A:

  • 2 hex collines (6 pts)
  • 2 hex rivière

B:

  • 3 hex collines (9pts)

Tour n°3

A: 7+2 B: 8+2

B

  • 2 hex rivière (6)
  • 3 hex désert (9)

Tour n°4

A:7+3 B:6

A:

  • 3 hex prairie (9) B: 2 hex rivière (6)

Tour n°5 A: 6+3 B: 4

A: 1 hex lac (9pts) B: 1 hex collines (3pts)

Second âge

Tour n°1

A: 6 B:10

B:

  • Create Race: Humains (6)
  • Create Race: Nains (6)
  • Create Subrace: Nains des collines (4)

A:

  • Create subrace: Humains (artraniens) (4)
  • Command race: Artraniens: fondation d'une ville: Artrain (4)
  • Advance City: Artrain: architecture remarquable (5)

Tour n°2

A:10+1 B:7+1

A:

  • Create Race: Halfelin (6)
  • Command Race: Halfelin: fondation ville: Firmanant (4)

B:

  • Create Race: Lézardins (6)

Tour n°3:

A:7+2 B:9+2

B:

  • Create Avatar: Humains: Maison Thelma (7)
  • Command Avatar: Fondation cité: Thélmain (1)

A:

  • Create Order: Lézardins: ordre des chevaliers serpents (6)
  • Command Race: Artraniens: fondation d'une ville: Arizain (4)

Tour n°4

A:8+3 B: 8+3

A

  • Command Race: Lézardins: alliance avec les nains des collines (4)
  • Command Chevaliers serpents: rassembler 1 armée (4)

B:

  • Command Race: Nains des collines: fonde une cité: Farnain (4)
  • Create Avatar: Humains: Maison Varide (7)
  • Command Avatar: Maison Varide: fondation Varidain

Tour n°5

A: 5+3 B:8+3

B:

  • Command City: Artrain: 1 armée (4)
  • Command City: Arizain: fortification (4) *Advance Civilization: Lézardins: art du combat (5)

A:

  • Create Avatar: Artrainiens: Le Général (7)
  • Command Avatar: Le Général: Fortification Artrain (4)

Troisième âge:

Tour n°1

A: 6+3 B: 7+3

B: *Command Race: Nains: fondation Therain

  • Command City: Therain: fortification
  • Command City: Therain: 1 armée

A:

  • Create Order: Halfelins: guilde d'explorateurs (4)
  • Command Race: Halfelins: fondation Prairain (2) *Command City: Prairain: fortification (2)

Tour n°2

A: 10+3 B: 9+3

A:

  • Command Order: Chevaliers serpents: 1 armée (2)
  • Command race: Lézardins: 1 armée (3)
  • Command City: Firmanant: université (2)
  • Advance Civilization: Humains: littérature (6)

B:

  • Command City: Farnain: 1 armée (2)
  • Command City: Varidain: 1 armée (2)
  • Command City: Thelmain: fortification (2)
  • Command Race: Humains: fondation cité: Sordain (3)
  • Command City:Sordain: fortification (2)
  • Command Race: Lézardins: 1 armée (3)
  • Create Order: Humains: compagnie mercenaire: Les épées du désert

Tour n°3

A: 6+3 B: 11+3

B:

  • Command Avatar: Le Général: 1armée (1)
  • Command City: Thelmain: 1 armée (2)
  • Command City: Sordain: 1 armée (2)
  • Command City: Sordain attaque Farnain (2)

9+2 Sordain: 1 armée, 1 compagnie mercenaire 8+2 Farnain: 2 armées Sordain gagne: Farnain est dévastée et ses armées détruites

  • Command Race: Lézardins attaque Sordain (3)

7+4 Lézardins: 2 armées, 1 ordre guerrier, + art du combat 8+1 Sordain: 1 fortification (armée éloignée) Lézardins gagnent: Sordain est dévastée, ses fortifications détruites

A:

  • Create order: Nains des collines: secte religieuse (4)
